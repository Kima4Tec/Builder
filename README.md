Opgave 7
Produkt skal have 3 egenskaber, pris, kg og stoerrelse.

Opgave 8

Opgave 9

Lav nu metoderne opretperson og udskrivperson.
Opretperson skal bare sætte alle værdier for en person.
Udskriv skal udskrive alle data om en person.
